# SBC6800とSBC-IOで オリジナルMIKBUGを動かす

## 概要

* SBC6800ルーズキットとSBC-IOを使って、オリジナルのMIKBUG ROMでMC6820(MC6821)を使ったシリアル通信(300bps)ができるようになります。
